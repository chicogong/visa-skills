## 核心知识库

### 关键规则

1. **日本使领馆不接受个人递签**，必须通过使馆指定的授权旅行社代办
2. 旅行社代办≠把信息全交给旅行社，核心材料必须申请人自己准备后交接
3. **领区严格**：按户籍地或有效居住证/社保所在地划分，跨领区申请极高风险

### 领区划分（大致）

| 领区 | 覆盖省市 | 主管使领馆 |
|---|---|---|
| 北京 | 北京、天津、河北、山西、内蒙古、黑龙江、吉林、辽宁、山东、河南 | 日本驻华大使馆 |
| 上海 | 上海、江苏、浙江、安徽 | 日本驻上海总领事馆 |
| 广州 | 广东、广西、福建、海南、云南、贵州 | 日本驻广州总领事馆 |
| 重庆 | 重庆、四川、西藏 | 日本驻重庆总领事馆 |
| 其他 | 各省对应领事馆 | — |

### 签证类型与年薪参考门槛

| 类型 | 年薪参考 | 每次最长停留 | 备注 |
|---|---|---|---|
| 单次 | 无硬性要求 | 15 天 | — |
| 三年多次 | ≈ 20 万元+ | 30 天 | 部分领区有高校毕业生弹性条件 |
| 五年多次 | ≈ 50 万元+ | 30-90 天 | 需对应个人所得税记录 |

> ⚠️ 以上为参考值，各领区细则和时期政策略有差异，以代办旅行社当时公告为准。

### 五大拒签 / 中止审查坑

1. **资金与税单不匹配** — 流水显示月收入 1 万，税单显示年薪 30 万，矛盾直接中止
2. **跨领区申请** — 未提供当地有效居住证/社保，直接退件
3. **申请表不规范** — 不可双面打印、二维码必须清晰、签名必须与护照尾页一致
4. **大额突击存入** — 临近申请前明显的大额转入，来源无法解释
5. **行程与假期矛盾** — 在职证明准假天数与计划行程天数不符

### 代办交接材料清单（申请人自备）

| 材料 | 说明 |
|---|---|
| 护照原件 | 有效期 6 个月以上，至少 2 页空白，含所有旧护照 |
| 身份证正反面复印件 | |
| 全家户口本整本复印件 | |
| 白底彩色证件照 | 45×45mm |
| **在职证明** | 公司抬头、公章、职位、年薪、准假日期 |
| 银行流水 | 工资卡近 3-6 个月，银行盖章版 |
| 纳税记录 / 存款证明 | 三年/五年多次申请必须 |
| 日本签证申请表 | 最新版 PDF，手写签名 |
| 个人信息处理同意书 | 使馆要求 |
| 行程单 + 机票/酒店预订 | 部分领区/类型要求 |
