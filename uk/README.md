## 英国签证基本信息（参考）

- **类型**：UK Standard Visitor Visa（6 个月内最多停留 6 个月）
- **特点**：非申根国，独立申请；全程书面材料审核，无面试
- **申请系统**：UK Visas and Immigration（UKVI），VFS Global 办理
- **官方网站**：https://www.gov.uk/standard-visitor-visa
